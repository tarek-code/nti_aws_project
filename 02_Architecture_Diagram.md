│  │  7. If cache miss → Forward to Primary ALB         │  │
│  │  8. Primary ALB → Web ASG                          │  │

│  │  12. CloudFront caches response                     │  │
│  │  13. User receives response                         │  │
│  └──────────────────────────────────────────────────────┘  │
│                                                              │
│  Failover Flow:                                             │
│  ┌──────────────────────────────────────────────────────┐  │
│  │  1. Primary ALB becomes unhealthy                    │  │
│  │  2. Route 53 health check fails (3 consecutive)     │  │
│  │  3. Route 53 switches to Secondary record           │  │

│  │  9. Spare Web ASG → Spare App ASG                   │  │
│  │  10. Spare App ASG → Spare RDS                      │  │
│  │  11. Response flows back                             │  │

│                                                              │
